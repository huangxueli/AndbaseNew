void com.ab.view.slidingmenu.SlidingListActivity.setBehindContentView ( View  view)
inline

Set the behind view content to an explicit view. This view is placed directly into the behind view 's view hierarchy. It can itself be a complex view hierarchy. When calling this method, the layout parameters of the specified view are ignored. Both the width and the height of the view are set by default to MATCH_PARENT. To use your own layout parameters, invoke setContentView(android.view.View, android.view.ViewGroup.LayoutParams) instead.

参数
viewThe desired content to display.
